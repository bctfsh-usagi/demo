# YakSok Demo

약속 프로젝트(웹 프로토타입)와 이후 실험 페이지들을 모아둔 테스트 저장소입니다.

- `index.html` : YakSok 메인 프로토타입
- `balmae.html` : 발매일기
- `tama.html` : Pastel Tamagotchi
- `liegame.html` : 거짓말 탐지 게임
- `minibrief.html` : MiniBrief 요약 서비스
- `pixelquest.html` : PixelQuest Ledger
- `blockbank.html` : 네오뱅크 목업 (실험 중)

정적 파일이므로 GitHub Pages 또는 Netlify Drop으로 곧바로 배포할 수 있습니다.
